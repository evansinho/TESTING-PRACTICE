# TESTING-PRACTICE
TDD practice with jest

## Built With

- jest
- JavaScript

### Prerequisites

- Node
- Npm

### Setup

- git clone `https://github.com/evansinho/TESTING-PRACTICE.git`
- cd TESTING-PRACTICE
- run `npm install`
- run `npm test`


### Usage

- Add test case if require and run `npm test`
